## Environnement d'exécution des TP 

Les TP sont faits pour être réalisés sur vos machines virtuelles Debian 12. Ils peuvent être réalisés sur d'autres linux (autres distributions ou versions ou MacOS). Si vous choisissez de les faire avec l'OS Windows il se pourrait que cela fonctionne tel quel mais aucune assurance n'est apportée. Ce n'est pas l'environnement préconisé. "*C'est à vos risques et périls.*"

La documentation des machines virtuelles de l'ISIMA => https://doc.isima.fr/my/vm/

Docker sur machine de TP de l'ISIMA peut rencontrer un bug noté dans la document de l'ISIMA => https://doc.isima.fr/pedagogie/bugs/#docker
